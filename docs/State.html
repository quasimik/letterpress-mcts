Class: State

State(ownership, playedWords, currentPlayer, legalPlays)

A class representing a game state. A game state in Letterpress consists of: 1) The ownership of the tiles 2) The words that have previously been played 3) The player to move next

Create a new game state.
Parameters:
Name Type Description

playedWords Array.<string> The array of played words.
currentPlayer number The player next to move.
legalPlays Array.<number> The array of legal moves from this game state, as move indexes.
