Class: Board

Board(letters, numRows, numCols, startingPlayer)

Class representing the Letterpress game board. Encapsulates the rules of Letterpress. Takes care of generating, storing, and returning legal moves for each given game state. Takes care of advancing a state given a legal move from that game state.
